04 / UI CONTRACTS

HTMLとJavaScriptをつなぐ印

印役割例
id1つの決まった要素を処理から取得する。searchInput、addPathDialog
data-*繰り返し生成される要素へ操作の意味を持たせる。data-path-details、data-add-path
aria-*支援技術とキーボード操作へ意味を伝える。aria-expanded、aria-keyshortcuts
CSSクラス見た目と、表示・非表示など一部の状態を表す。hidden、active
