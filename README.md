# Path Catalog

フォルダパスやURLを「探す・貼る」作業を、最短にするためのローカルHTMLビューアです。  
**カテゴリで整理して一覧化し、クリック一発でクリップボードへコピー**できます。  
ビルド不要・サーバー不要で、`path-catalog.html` を開くだけで動きます。

![Path Catalog sample](docs/path-catalog-sample-capture.png)

---

## これは何を解決するツールか

日常業務で、こんな状況になっていませんか？

- 環境URL、申請先、共有フォルダ、手順書リンクが散らばっていて「どこだっけ？」が頻発する
- Teams / Slack / メール / Excel / 個人メモに点在し、引き継ぎやチーム共有が難しい
- パスやURLを毎回探してコピペするだけで、地味に時間が溶ける
- 「正しいリンク」が更新されても、周知が追いつかず古いリンクが残る

Path Catalog は、これらを **「リンク・パスのカタログ化」** によって解消します。

- 情報は `data.js` に集約（= 共有・レビューしやすい）
- 画面は「見つける」ことに特化（= 迷わず辿れる）
- 操作は「クリックしてコピー」だけ（= 手順が短い）

---

## 向いている利用シーン

- チームで使う **環境URLランチャー**（商用 / 検証 / ステージング など）
- 各種申請・運用で使う **共有フォルダ / テンプレ / 手順書の集約**
- システム受け入れ側の業務でありがちな **参照先の多さの整理**
- 新メンバーのオンボーディングで「まずここ見て」を作る用途

---

## 特長

- **ゼロセットアップ**：ブラウザで `path-catalog.html` を開くだけ（静的HTML）
- **整理しやすい構造**：タブ → カテゴリ → サブカテゴリ
- **コピーが速い**：行クリックで即クリップボードへコピー
- **メモ対応**：補足ノートをホバーで表示
- **データ編集が簡単**：`data.js` を編集するだけで即反映
- **視認性重視UI**：カテゴリごとに色分けされたカード表示

---

## 3分で使い始める（Quick Start）

1. リポジトリをダウンロードまたは clone
2. `path-catalog.html` をブラウザで開く
3. 左のサイドバーからカテゴリを選択
4. 目的の行をクリックしてコピーし、貼り付け

---

## 使い方

- **左**：カテゴリ選択
- **右**：カテゴリ / サブカテゴリ / パス（URL）の一覧
- **クリック**：クリップボードへコピー
- **ノート**：行末の `i` にホバーで表示

## データの編集（data.js）

表示するデータは `data.js` の `window.PATH_CATALOG_DATA` に定義します。  
記述方法は 2 通りあります。

### 方式1：フラット形式（1行 = 1アイテム）

最も直感的で、差分レビューもしやすい形式です。

`` + `js
const r = String.raw;
window.PATH_CATALOG_DATA = [
  {
    tab: "共通",
    category: "申請",
    subCategory: "経費",
    name: "経費精算フォルダ",
    path: r`\\fileserver\share\finance\expense`,
    note: ""
  }
];
`` + `

---

### 方式2：階層形式（タブ > カテゴリ > サブカテゴリ > アイテム）

情報のまとまりを重視したい場合に向いています。  
定義したデータは `flatten(DATA)` でフラット化して利用します。

`` + `js
const r = String.raw;

const DATA = {
  tabs: [
    {
      name: "共通",
      categories: [
        {
          name: "申請",
          subCategories: [
            {
              name: "経費",
              items: [
                {
                  name: "経費精算フォルダ",
                  path: r`\\fileserver\share\finance\expense`,
                  note: ""
                }
              ]
            }
          ]
        }
      ]
    }
  ]
};

window.PATH_CATALOG_DATA = flatten(DATA);
`` + `

---

### 編集のコツ

- Windows の `\\server\share` のようなパスは `String.raw` を使うと安全
- 値が空でも動作します（未指定項目は自動的に整理されます）

---

## 注意点（スコープ）

- 本ツールは **リンク・パスの整理とコピー**に特化しています
- 認証情報・パスワード管理は対象外です
- モバイル表示には対応していません（PC利用前提）
