## Path Catalogでできること

| 整理する | すぐ使う | 安心して保存する |
| :---: | :---: | :---: |
| タブ・カテゴリ・サブカテゴリで、パスやURLを見やすく分類 | 必要なパスをクリックするだけでクリップボードへコピー | ブラウザへ自動保存し、`data.js`でバックアップ・移行 |

## すぐに始める

1. `path-catalog.html`と`data.js`を同じフォルダーに置く
2. `path-catalog.html`をブラウザで開く
3. 右上の`＋ 追加`から、よく使うパスを登録する

これだけで使い始められます。

## 操作早見表

| やりたいこと | 操作 |
| --- | --- |
| パスをコピー | 一覧のパスをクリック |
| パス・カテゴリ・タブを追加 | 右上の`＋ 追加` |
| サブカテゴリを追加 | カテゴリ右側の`＋` |
| 選んだサブカテゴリへパスを追加 | サブカテゴリ右側の`＋` |
| ノートを確認 | 右端の`i`にマウスを重ねる |
| タブを切り替える | `← / →`キー |
| カテゴリを切り替える | `↑ / ↓`キー |

## データを守る

登録内容は、使用中のブラウザへ自動保存されます。

| 操作 | 用途 |
| --- | --- |
| `データ管理` → `データを書き出す` | 現在の内容を`data.js`としてバックアップ |
| `データ管理` → `データを読み込む` | バックアップや別ブラウザのデータを復元 |

配布用の初期データを更新するときは、書き出したファイルで既存の`data.js`を置き換えます。

## 動作環境

- 最新のChrome、Edge、Firefox、Safari
- デスクトップ画面を推奨
- スタイルとフォントの表示にはインターネット接続が必要
