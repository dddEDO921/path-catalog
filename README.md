# Path Catalog

よく使うフォルダーパスやURLを整理して、すぐにコピーできるツールです。

![Path Catalog sample](docs/path-catalog-sample-capture.png)

## できること

- パスやURLをタブ・カテゴリ・サブカテゴリで整理
- 一覧をクリックしてコピー
- ブラウザ上からデータを追加
- `data.js`でバックアップ・移行

## 始め方

1. `path-catalog.html`と`data.js`を同じフォルダーに置く
2. `path-catalog.html`をブラウザで開く

インストールやビルドは不要です。

## 基本操作

- タブや左側のカテゴリを選ぶと、表示内容が切り替わります。
- パスをクリックすると、クリップボードへコピーされます。
- `i`にマウスを重ねると、ノートが表示されます。

### データを追加する

- 右上の`＋ 追加`: パス・カテゴリ・タブを追加
- カテゴリ右側の`＋`: サブカテゴリを追加
- サブカテゴリ右側の`＋`: その場所へパスを追加

### キーボード操作

- `← / →`: タブを切り替える
- `↑ / ↓`: カテゴリを切り替える

## データの管理

追加した内容は、使用中のブラウザに自動保存されます。

- `データ管理` → `データを書き出す`: 現在の内容を`data.js`として保存
- `データ管理` → `データを読み込む`: 保存した`data.js`を読み込む

別のブラウザへ移行するときや、バックアップを取るときに利用してください。

配布用の初期データを更新する場合は、書き出したファイルで既存の`data.js`を置き換えます。

## 動作環境

- 最新のChrome、Edge、Firefox、Safari
- デスクトップ画面を推奨
- スタイルとフォントの表示にはインターネット接続が必要
